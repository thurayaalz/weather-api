# Weather App

A simple weather application built with vanilla JavaScript.  
It fetches real-time weather data from an external API and displays the current temperature and location.

## Features
- Search for any city
- Fetch live weather data
- Clean modular structure (API, UI, elements)
- Bundled with Webpack

## Tech Stack
- JavaScript (ES6 modules)  
- Webpack  
- HTML / CSS  
- Weather API (Visual Crossing)
